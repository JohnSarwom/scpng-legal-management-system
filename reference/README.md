# Legal Module Reference

This folder documents the main prototype changes made during the UI refinement pass.

Use these notes when editing the app so related behavior stays consistent across pages.

## Files

- [dashboard-welcome.md](dashboard-welcome.md) - personalized dashboard hero and quick actions.
- [hybrid-filters.md](hybrid-filters.md) - compact filter bars, popovers, and active chips.
- [search-behavior.md](search-behavior.md) - text search matching for IDs, numbers, titles, and subjects.
- [role-name-mapping.md](role-name-mapping.md) - role selector to real SCPNG staff names.
- [relationship-map.md](relationship-map.md) - interactive relationship map nodes, tooltips, and navigation.
- [case-access-control.md](case-access-control.md) - case ownership, grants, confidentiality, and per-person read/write access.
- [verification.md](verification.md) - useful build and browser checks.
